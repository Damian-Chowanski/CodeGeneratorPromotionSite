This is the screen where you can edit the list of previously generated codes. You can set whether a code is a winner or loser, and check if it has already been used or is still eligible for winning.

If you want to edit a code, click on the one you've selected, and then make the changes in the fields available below the table. You have three buttons at your disposal:

  • Change - Allows you to overwrite the selected code.
  • Add - Lets you add a new code to the list
  • Remove - Deletes the selected code from the list.

    • After finishing the modification of codes, select the back button to test the codes you've created.
