The first scene that launches with the application is a screen designed for code generation. On this screen, the user is prompted to generate codes that will be used for further gameplay.

The user can generate as many codes as they wish, but there must be a minimum of 3 codes. In the next field, the user is asked to specify the number of characters that the codes should consist of.

The generated codes have three values:

  • Code - a random string of characters
  • isUsed - This tells you if the code has already been claimed by another user.
  • isWon - Is this code a winner?

    • The generated codes are displayed in a table. Only the codes that are won and unused provide the user with a cash injection.

    After generating codes you can click on Administration Panel to edit them or PLAY! to start the game!
